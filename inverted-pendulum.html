Inverted Pendulum
Dynamic Control System

C++ | State-Space Control | Sensor Fusion | Stepper Motors

Implemented a C++ state-space control system with eigenvalue analysis to stabilize a highly dynamic physical system, integrating a sensor fusion pipeline that processes high-speed rotary encoder data into high-frequency PID loops.

Executed full hardware integration of stepper motors and sensors, systematically debugging signal noise to achieve reliable real-time performance directly applicable to PLC feedback control in production automation.
